# Cover Images

Each logical cover exists in two storage variants because the final proposal compares full branches, not only embedding rules.

## Variants

- `spatial/{source}/`
  - grayscale PNG inputs for sequential LSB replacement
- `frequency/{source}/`
  - grayscale JPEG Q=95 inputs for DCT-LSB

## Sources

- `real`
- `ml_a` (`SDXL 1.0`)
